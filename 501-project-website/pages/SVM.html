SVM

Support Vector Machine is supervised machine learning algorithm that is widely used in classification and regression. It works with binary labeled datasets by representing the datapoints as points in a space, and then separate the datapoints with different labels widely using the mappings of the points in the space. This way, having the new datapoints the mappings of them in the same space will be used to predict which label the new points belong to.

SVM is suitable for classifying texts and hypertexts, images, as well as identifying handwritten notes, etc. Sometimes it is even used for very complicated job, such as classifying different proteins in medical science. For this project, it will be used to classify the twitter texts.

Text Dataset

Class distribution

The goal of using SVM on the text dataset is to find out whether the SVM model is able to classify and predict new text data based on training. In the data cleaning part, two text dataset using different keywords "school" and "education" were cleaned and put together as one labeled dataset. The distribution of these two class labels: school: 678, education: 610.

Baseline model for comparison

As a baseline comparison, a random classifier on the data was performed. This step carried out a uniform random number generation with random numbers sampled based on the distribution of labels in dataset (y labels). The reported accuracy of the random classifier is 0.5, with recall values 0.53 and f-score 0.47.

Feature selection

In order to perform support vector machine models on the dataset, the text data with 1288 entries will be converted into an onehoy matrix. After transformation the resulted X matrix is a sparse matrix consists of 1288 rows and 643 columns.

Model tuning

Model tuning is the experimental process of finding the optimal values of hyperparameters to maximize model performance. To perform model tuning for SVM classifiers, first for linear model will be tested for hyperparameter, and then several different kernels will be tested.
