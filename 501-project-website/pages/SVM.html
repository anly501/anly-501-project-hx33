SVM

Support Vector Machine is supervised machine learning algorithm that is widely used in classification and regression. It works with binary labeled datasets by representing the datapoints as points in a space, and then separate the datapoints with different labels widely using the mappings of the points in the space. This way, having the new datapoints the mappings of them in the same space will be used to predict which label the new points belong to.

SVM is suitable for classifying texts and hypertexts, images, as well as identifying handwritten notes, etc. Sometimes it is even used for very complicated job, such as classifying different proteins in medical science. For this project, it will be used to classify the twitter texts.
